# WEBSnake — Интерактивная веб-игра «Змейка» с клиент-серверной архитектурой

Классическая игра «Змейка», реализованная на чистом JavaScript (Vanilla JS) с полноценным бэкендом на Node.js для сбора игровой статистики, авторизации пользователей и вывода глобального лидерборда.

![WEBSnake](./WEBSnake_screen.png)

## 🚀 Технологический стек
* **Frontend:** JavaScript (ES6+), HTML5, CSS3 (Модульная структура стилей)
* **Backend:** Node.js, Express (REST API)
* **Database:** SQL / СУБД (Хранение игровых сессий, аккаунтов и очков)

## ✨ Инженерные и архитектурные решения
* **Модульная структура фронтенда:** Логика приложения разбита на изолированные компоненты: `game/` (игровой движок и рендеринг), `form/` (интерфейсы входа и регистрации), `statistic/` (вывод лучших результатов).
* **Игровой цикл (Game Loop):** Реализация динамического изменения скорости игры, точный расчет координатной сетки и обработка коллизий (столкновение змейки со стенами и собственным хвостом).
* **Сохранение состояния (State):** Асинхронная отправка результатов игровой сессии на бэкенд сразу после окончания игры (Game Over) для формирования глобального рейтинга.

## 📁 Структура проекта

```text
.
├── database/           # Скрипты инициализации и запросов к базе данных
├── game/               # Логика игрового процесса (движение, еда, коллизии)
├── form/               # Компоненты форм регистрации и авторизации пользователей
├── statistic/          # Модуль отображения игровой статистики и рекордов
├── styles/             # Глобальные и модульные CSS стили проекта
├── scripts/            # Вспомогательные JS утилиты
├── image/              # Графические ассеты игры
├── index.html          # Главная точка входа приложения
├── server.js           # Бэкенд-сервер на Node.js (REST API)
└── package.json        # Манифест зависимостей проекта
```

## ⚙️ Локальный запуск

1. Установите зависимости бэкенда:
   ```bash
   npm install
   ```
2. Запустите сервер:
   ```bash
   node server.js
   ```
3. Откройте проект в браузере.
